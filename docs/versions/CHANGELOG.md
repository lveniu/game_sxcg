## 版本约定

- **MAJOR** (主版本) - 不兼容更新
- **MINOR** (次版本) - 功能增加（向后兼容）
- **PATCH** (补丁) - Bug 修复

## 标签约定

- `Added` - 新功能
- `Changed` - 现有功能变更
- `Deprecated` - 即将移除的功能
- `Removed` - 已移除的功能
- `Fixed` - Bug 修复
- `Security` - 安全相关
