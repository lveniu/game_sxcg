## 五、IP/创意：骰子驱动Roguelike的独特性

### 核心创意的独特价值

**"骰子驱动Roguelike"不是缝合，而是化学反应：**

| 维度 | 普通Roguelike | 骰子驱动Roguelike（我们） |
|------|--------------|------------------------|
| 策略入口 | 选卡/选装备 | 掷骰+重摇+卡牌联动 |
| 随机性 | 被动接受 | 主动博弈（重摇决策） |
| 爽感来源 | 组合成型 | 摇出神仙组合+看它碾压 |
| 失败体验 | "我选错了" | "差一点就是三条！"（赌狗遗憾） |
| 上瘾模型 | 成长感 | 赌博心理+成长感（双驱动） |

### 小游戏场景迁移能力

**骰子系统天然适合小游戏的理由：**

1. **单局短** — 每关2-3分钟，完美匹配碎片时间
2. **操作简单** — 掷骰子+点卡牌，触屏友好
3. **视觉冲击** — 骰子滚动+组合特效+伤害数字=强短视频素材
4. **社交货币** — "我摇出了三条！"比"我抽到了SSR"更有话题性
5. **留存设计** — "差一点就出顺子"的遗憾感=天然留存驱动力
6. **付费点** — 额外重摇次数、骰子皮肤、特殊骰子面=天然商业化接口

### 竞品差异化

| 竞品方向 | 代表作 | 我们的差异 |
|---------|--------|-----------|
| 纯骰子 | Dicey Dungeons | 我们有自走棋战斗+卡牌+站位 |
| 纯自走棋 | 金铲铲/自走棋 | 我们有骰子驱动的赌狗体验 |
| 纯Roguelike | 杀戮尖塔/小丑牌 | 我们融合了骰子+自走棋+卡牌 |
| 小游戏Roguelike | 各类微信小游戏 | 我们有"骰子×卡牌联动"的核心创新 |
