# 游戏规划文档 — GameSXCG

## 1. 游戏概述

| 项 | 内容 |
|----|------|
| **项目名** | GameSXCG |
| **引擎** | Unity 2022.3 LTS |
| **目标平台** | PC / 移动端 |
| **游戏类型** | Roguelike + 骰子 + 卡牌构筑 + 自走棋 |
| **单局时长** | 15~30 分钟 |
| **核心体验** | 掷骰子赌组合 → 组合在战场爆炸 → 卡牌放大/保底 → 看自动战斗碾压 |

## 2. 核心循环（单局）

```
[开始] → 选择初始单位/英雄
    ↓
[事件节点] → 战斗 / 商店 / 随机事件 / 精英 / Boss
    ↓
[骰子阶段] → 掷骰子 → 查看组合 → 决定是否重摇
    ↓
[出牌阶段] → 根据骰子结果召唤/打出卡牌（卡牌与骰子联动）
    ↓
[站位阶段] → 调整3个单位在棋盘上的位置（前/中/后排）
    ↓
[即时战斗] → 自走棋式自动战斗，骰子组合效果在战场爆发
    ↓
[战斗结算] → 获得金币/卡牌/骰子面升级 → 回到事件节点选择下一关
    ↓
[循环直到] → 通关或团灭
```

## 3. 核心机制详解

### 3.1 骰子系统

**核心定位**：骰子组合是战斗伤害的主要来源（70%+），卡牌和站位均为辅助。

**基础机制**：
- 玩家每局拥有 `M` 个骰子，每个骰子 `N` 个面（默认六面 1~6，可扩展面数）
- 战斗前掷出所有骰子，获得一组点数池
- **重摇机制**：每局免费重摇 2 次，是心理博弈的核心（见好就收 vs 赌一把大的）
- **重摇卡**：可消耗骰子点数获得额外重摇次数

**组合奖励（扑克算法）**：

| 组合 | 条件 | 战斗效果 | MVP |
|------|------|----------|-----|
| 三条 | 三个相同点数 | 全体攻击+20% | ✓ |
| 顺子 | 三个连续点数 | 全体攻速+20% | ✓ |
| 对子+单点 | 两个相同+一个不同 | 单点对应英雄获得专属Buff | ✓ |
| 葫芦 | 两对 | 全体暴击率+15% | Alpha |
| 四条 | 四个相同 | 单体秒杀技能 | Alpha |
| 同花顺 | 同色且连续 | 终极清场技能 | Alpha |

- **骰子可成长**：通过奖励把某一面改成特殊效果（如「6→闪电」、3→冰冻）
- **骰子可合成**：两个相同骰子合并成一个高级骰子（更多面/更高数值）

**赌狗心理设计**：
- 第一次掷骰后，玩家看到"差1点就是顺子"，产生重摇冲动
- 即使骰子很烂，保底机制（属性卡）确保不会完全无法战斗
- 每局都有"神之一手"的可能：低属性+好骰子能翻盘

### 3.2 卡牌系统

**核心定位**：卡牌是骰子的"辅助系统"，不能强过骰子。属性卡保底，战斗卡放大，进化卡是赌狗大奖。

- **属性卡牌**（保底系统）：
  - *力量训练*：本局永久+3攻击，骰子烂时堆攻击
  - *坚固护甲*：本局永久+3防御，骰子烂时堆防御
  - *灵敏训练*：本局永久+2速度，骰子烂时堆速度
  - 叠加成长，整局生效

- **战斗卡牌**（放大器，与骰子联动）：
  - *斩击*：消耗1点。**对子+斩击**：伤害翻倍
  - *重摇*：消耗1点换1次重摇机会
  - *护盾冲击*：消耗2点。**三条+护盾冲击**：护盾量+50%
  - *寻找弱点*：消耗1点。**顺子+寻找弱点**：暴击率额外+20%
  - 本场战斗生效，每场可打

- **进化卡牌**（赌狗大奖）：
  - *进化觉醒*：消耗3个相同点数，解锁该职业进化形态
  - 进化后英雄属性大幅提升，技能增强

- 卡牌稀有度（白/蓝/紫/金），同名卡牌2合1升级
- 卡组上限 10 张，战斗前可调整

### 3.3 自走棋战斗

**核心定位**：自动战斗是"展示骰子效果的舞台"，玩家观战的核心是看"我摇出的组合如何碾压敌人"。

**战场规则**：
- 固定 3×4 棋盘（MVP），我方最多 3 个单位，敌方最多 3 个单位
- 简化站位逻辑（不做复杂克制）：
  - *前排*：吸收伤害，骰子护盾效果+30%（适合坦克）
  - *中排*：标准输出，无增减（适合射手/刺客）
  - *后排*：骰子伤害+20%，但被突进单位克制（适合射手）
- 站位决策 5 秒内完成，不烧脑

**战斗演出**：
- 战斗时长：15~30 秒
- 玩家不可操作，可 2x 加速
- 视觉重点：
  1. 战斗开始时，骰子飞入战场中央，播放组合特效（三条=金光、顺子=疾风）
  2. 英雄攻击时，伤害数字弹出，带有骰子组合图标
  3. 进化觉醒触发时，全屏特效+英雄变身动画
- 伤害数字要夸张、弹出要快、特效要明显
- 如果骰子组合很差，战斗会艰难甚至失败，强化"下一局一定要摇出好组合"的驱动力

**后续扩展（Alpha）**：
- 棋盘格子 Buff：增益格/特殊格/陷阱格
- 更大棋盘（4×4或更大）
- 特殊地形效果（低地、高地、障碍物）

### 3.4 单位系统
- 每个单位有：职业 + 属性成长 + 技能槽 + 骰子空位
- 进化分为**多条分支路线**：
  - *例如*：战士 → 守护者 / 狂战士 / 骑士
  - *法师* → 元素使 / 冰霜法师 / 召唤师
  - *物理输出* → 刺客 / 射手 / 狂战士
- **进化条件**：消耗进化卡牌 + 达到等级/属性阈值
- **融合变身**（后续扩展）：两个进化到终端的单位可以融合成更强大的终极体
- 场上最多 5~6 个单位

## 4. 关卡/节点设计

| 节点类型 | 说明 |
|---------|------|
| 普通战斗 | 标准敌人，低风险 |
| 精英战斗 | 更强敌人，奖励更好 |
| 商店 | 用金币买卡牌、骰子面、单位 |
| 事件 | 随机选择（增益/减益/特殊奖励） |
| 篝火 | 恢复/强化一个单位 |
| Boss | 每章终点，通关解锁下一章 |

## 5. 成长体系（局外 Meta）

- **天赋树**：解锁新骰子面、初始金币、卡牌掉落率等
- **图鉴**：收集单位进化路线，解锁新英雄
- **难度等级**：通关后解锁更高难度

## 6. UI/UX 规范（待细化）

- 战斗准备界面：左侧骰子区，中间卡牌区，右侧单位/站位区
- 战斗过程：俯视/斜45°视角，技能特效要夸张但清晰
- 结算界面：数据可视化（伤害统计、进化路线回顾）

## 7. 技术要求

- 支持多分辨率（PC 窗口化 + 手机竖屏/横屏）
- 卡牌数据 JSON 驱动（方便平衡性调整）
- 战斗回放/加速功能
- 存档系统（局内中断可恢复）

## 8. 开发里程碑

| 阶段 | 内容 | 预估时间 |
|------|------|---------|
| **MVP** | 骰子+卡牌+单单位战斗原型 | 2~3 周 |
| **Alpha** | 完整单局循环（事件节点→Boss） | 3~4 周 |
| **Beta** | 多英雄、进化系统、局外成长 | 4~6 周 |
| **Release** | 美术包装、音效、平衡性调优 | 3~4 周 |
