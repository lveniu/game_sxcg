# 游戏规划文档 — GameSXCG

## 1. 游戏概述

| 项 | 内容 |
|----|------|
| **项目名** | GameSXCG |
| **引擎** | Unity 2022.3 LTS |
| **目标平台** | PC / 移动端 |
| **游戏类型** | Roguelike + 骰子 + 卡牌构筑 + 自走棋 |
| **单局时长** | 15~30 分钟 |
| **核心体验** | 掷骰子 → 出牌改属性/进化 → 看单位自动战斗 |

## 2. 核心循环（单局）

```
[开始] → 选择初始单位/英雄
    ↓
[事件节点] → 战斗 / 商店 / 随机事件 / 精英 / Boss
    ↓
[战斗前准备] → 掷骰子 → 获得 N 个骰子点数
    ↓
[出牌阶段] → 打出卡牌，消耗骰子点数，改变单位属性或进化方向
    ↓
[即时战斗] → 自走棋式自动战斗，单位按配置好的属性/技能互殴
    ↓
[战斗结算] → 获得金币/卡牌/骰子面升级 → 回到事件节点选择下一关
    ↓
[循环直到] → 通关或团灭
```

## 3. 核心机制详解

### 3.1 骰子系统
- 玩家每局拥有 `M` 个骰子，每个骰子 `N` 个面（默认六面 1~6，可扩展面数）
- 战斗前掷出所有骰子，获得一组点数池
- **重摇机制**：每局可重摇次数有限，用完后可观看广告获得额外重摇次数
- **组合奖励**（扑克算法）：
  - *同花* — 所有骰子结果同色/同属性，奖励临时攻击加成
  - *顺子* — 点数连续（5个以上），奖励闪避或移动速度
  - *同花顺* — 同色且连续，奖励技能冷却缩短或暴击
  - *四条* / *满堂红* — 多个相同点数，奖励元素伤害或护盾
  - *三条带顺* — 三个相同点数+一个顺子，奖励全属性提升
- **骰子可成长**：通过奖励把某一面改成特殊效果（如「6→闪电」、"3→冰冻"）
- **骰子可合成**：两个相同骰子合并成一个高级骰子（更多面/更高数值）

### 3.2 卡牌系统
- **属性卡牌**：
  - *养成属性*：永久提升单位基础属性（生命、攻击、防御、速度），可叠加成长
  - *战斗卡牌*：消耗骰子点数，提供临时 Buff 或改变骰子结果（如「强制重摇」「面值+1」「同花转换」）
- **成长道具卡牌**：不消耗骰子，直接给单位增加经验值、解锁新技能槽、扩展骰子面数
- **进化卡牌**：消耗特定点数组合，解锁单位进化路线的分支节点
- 卡牌有稀有度（白/蓝/紫/金）
- 卡组上限 10 张，战斗前可调整

### 3.3 自走棋战斗
- 战场格子可调节（默认 3×4 或 4×4，后续支持更大棋盘）
- 每个棋盘格子可能提供场地 Buff：
  - *增益格*：攻击/防御/治疗加成
  - *特殊格*：元素伤害增加、技能冷却缩短、护盾回复
  - *陷阱格*：敌方专用，我方单位站上受到减益
- 单位自动寻敌、释放技能
- 战斗不可操作，纯看之前的构筑成果

### 3.4 单位系统
- 每个单位有：职业 + 属性成长 + 技能槽 + 骰子空位
- 进化分为**多条分支路线**：
  - *例如*：战士 → 守护者 / 狂战士 / 骑士
  - *法师* → 元素使 / 冰霜法师 / 召唤师
  - *物理输出* → 刺客 / 射手 / 狂战士
- **进化条件**：消耗进化卡牌 + 达到等级/属性阈值
- **融合变身**（后续扩展）：两个进化到终端的单位可以融合成更强大的终极体
- 场上最多 5~6 个单位

## 4. 关卡/节点设计

| 节点类型 | 说明 |
|---------|------|
| 普通战斗 | 标准敌人，低风险 |
| 精英战斗 | 更强敌人，奖励更好 |
| 商店 | 用金币买卡牌、骰子面、单位 |
| 事件 | 随机选择（增益/减益/特殊奖励） |
| 篝火 | 恢复/强化一个单位 |
| Boss | 每章终点，通关解锁下一章 |

## 5. 成长体系（局外 Meta）

- **天赋树**：解锁新骰子面、初始金币、卡牌掉落率等
- **图鉴**：收集单位进化路线，解锁新英雄
- **难度等级**：通关后解锁更高难度

## 6. UI/UX 规范（待细化）

- 战斗准备界面：左侧骰子区，中间卡牌区，右侧单位/站位区
- 战斗过程：俯视/斜45°视角，技能特效要夸张但清晰
- 结算界面：数据可视化（伤害统计、进化路线回顾）

## 7. 技术要求

- 支持多分辨率（PC 窗口化 + 手机竖屏/横屏）
- 卡牌数据 JSON 驱动（方便平衡性调整）
- 战斗回放/加速功能
- 存档系统（局内中断可恢复）

## 8. 开发里程碑

| 阶段 | 内容 | 预估时间 |
|------|------|---------|
| **MVP** | 骰子+卡牌+单单位战斗原型 | 2~3 周 |
| **Alpha** | 完整单局循环（事件节点→Boss） | 3~4 周 |
| **Beta** | 多英雄、进化系统、局外成长 | 4~6 周 |
| **Release** | 美术包装、音效、平衡性调优 | 3~4 周 |
