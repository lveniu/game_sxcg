1. **🔴 首包4MB** — 第1周末必须做一次空项目WebGL构建，摸底体积基线
2. **🟡 工期紧张** — 90人天 vs 80人天可用，P2/P3任务可延后，建议+1人
3. **🟡 机制怪** — MVP先做3种（护盾互换/伤害反弹/狂暴），其余v1.1补
4. **🟢 美术** — 开发阶段用占位图，核心玩法完成后再替换
