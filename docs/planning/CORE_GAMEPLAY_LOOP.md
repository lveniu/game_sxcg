# 核心循环设计 — 赌狗流（Dice-Driven）

> 版本：v1.0  
> 定位：本游戏以**骰子赌狗快感**为第一爽感，卡牌构筑与站位策略为辅助层次。

---

## 1. 核心爽感公式

```
紧张（掷骰） → 期待（看重摇） → 释放（组合爆炸） → 满足（看自动战斗碾压）
```

- **70% 爽感来自骰子**：摇出三条/顺子/同花顺等组合，直接转化为战场伤害/特效
- **20% 爽感来自卡牌**：用卡牌放大骰子效果，或在骰子烂时兜底
- **10% 爽感来自站位**：简单的前中后排站位放大骰子收益

---

## 2. 单局核心循环

```
[开局] → 选初始英雄（带1个基础骰子面）
    ↓
[骰子阶段] → 掷3骰 → 看组合 → 决定是否重摇（免费2次）
    ↓
[出牌阶段] → 召唤英雄（消耗点数）→ 打属性卡/战斗卡（与骰子联动）
    ↓
[站位阶段] → 调整3个单位在3×4棋盘上的位置（前排承伤/中排输出/后排增伤）
    ↓
[自动战斗] → 15~30秒观战，骰子组合效果在战场爆发
    ↓
[结算] → 获得金币/卡牌/骰子面升级
    ↓
[循环] → 普通战 / 精英战 / 商店 / 事件 / Boss
```

**单局时长**：15~30 分钟  
**每关循环**：约 2~3 分钟（骰子+出牌 30秒，战斗 20秒，结算 10秒）

---

## 3. 各阶段详解

### 3.1 骰子阶段（战前）— 绝对核心

**机制**：
- 3 个六面骰（1~6），每场战斗前重置投掷
- 免费重摇 2 次（可逐颗重摇或全重摇）
- 重摇卡可消耗骰子点数换取额外重摇次数

**组合奖励（MVP简化版）**：

| 组合 | 条件 | 战斗效果 |
|------|------|----------|
| 三条 | 三个相同点数 | 全体攻击+20% |
| 顺子 | 三个连续点数 | 全体攻速+20% |
| 对子+单点 | 两个相同+一个不同 | 单点对应的英雄获得专属Buff |

> Alpha 阶段扩展：葫芦、四条、同花顺等高级组合，效果更爆炸。

**赌狗心理设计**：
- 第一次掷骰后，玩家看到"差1点就是顺子"，产生重摇冲动
- 重摇机制是心理博弈核心：见好就收 vs 赌一把大的
- 即使骰子很烂，保底机制（属性卡）确保不会完全无法战斗

---

### 3.2 出牌阶段 — 辅助骰子

**召唤英雄**：
- 消耗骰子点数召唤英雄上场（坦克2点/射手2点/刺客1点）
- 召唤后的骰子点数不再用于其他效果
- 人口上限 3，超出需先下场单位

**卡牌与骰子联动（重构后）**：

| 卡牌 | 类型 | 与骰子联动 |
|------|------|-----------|
| 力量训练 | 属性 | 无联动，纯保底（骰子烂时堆攻击） |
| 坚固护甲 | 属性 | 无联动，纯保底 |
| 灵敏训练 | 属性 | 无联动，纯保底 |
| 斩击 | 战斗 | **对子+斩击**：伤害翻倍 |
| 护盾冲击 | 战斗 | **三条+护盾冲击**：护盾量+50% |
| 寻找弱点 | 战斗 | **顺子+寻找弱点**：暴击率额外+20% |
| 重摇 | 战斗 | 消耗1点换1次重摇机会 |
| 进化觉醒 | 进化 | 消耗三个相同点数，解锁进化形态 |

**设计意图**：
- 属性卡是"保底系统"：骰子烂时，叠属性卡也能打
- 战斗卡是"放大器"：好骰子+好卡牌=爆炸输出
- 进化觉醒是"赌狗大奖"：摇出三条本身就是爽点，再解锁进化=爽上加爽

---

### 3.3 站位阶段 — 放大骰子效果

**简化站位逻辑**（不做复杂克制）：

| 位置 | 效果 | 适合单位 |
|------|------|----------|
| 前排 | 吸收伤害，骰子护盾效果+30% | 坦克 |
| 中排 | 标准输出，无增减 | 射手/刺客 |
| 后排 | 骰子伤害+20%，但被突进单位克制 | 射手 |

**设计意图**：
- 站位不是烧脑策略，而是"如何把骰子收益最大化"的简单选择
- 3 个单位、3×4 棋盘，站位决策 5 秒内完成
- 后续可扩展：特定格子 Buff（Alpha 阶段）

---

### 3.4 自动战斗 — 展示骰子效果

**战斗时长**：15~30 秒  
**玩家操作**：无（纯观战，可加速 2x）

**视觉重点**：
1. 战斗开始时，骰子飞入战场中央，播放组合特效（三条=金光、顺子=疾风）
2. 英雄攻击时，伤害数字弹出，带有骰子组合图标
3. 进化觉醒触发时，全屏特效+英雄变身动画

**观战体验设计**：
- 玩家不是在"看AI打架"，而是在"看我摇出的组合如何碾压敌人"
- 伤害数字要夸张、弹出要快、特效要明显
- 如果骰子组合很差，战斗会艰难甚至失败，强化"下一局一定要摇出好组合"的驱动力

---

### 3.5 结算阶段 — 赌狗的奖励

**战胜奖励**（三选一）：
1. **新骰子面**：把某个骰子的某一面替换成特殊效果（如「6→闪电伤害」）
2. **新卡牌**：随机获得1张属性卡或战斗卡
3. **金币**：用于商店购买

**失败处理**：
- 根据已通关关卡数给予临时奖励（安慰奖）
- 保留已获得的骰子面升级和卡牌（Roguelike 累积感）
- 显示"差 X 点就是三条/顺子"，制造遗憾感，驱动再开一局

## 4. 爽感曲线（单局内）

| 阶段 | 情绪 | 设计支撑 |
|------|------|----------|
| 开局 | 紧张+期待 | 骰子决定本局起点，不知道会摇出什么 |
| 前期（1-3关） | 摸索+小爽 | 熟悉组合效果，偶尔摇出对子/三条 |
| 中期（4-6关） | 成长感 | 属性卡叠起来，骰子面开始升级 |
| 后期（7-9关） | 碾压感 | 好组合+高属性=爆炸输出，一路平推 |
| Boss战 | 高潮 | 所有累积在此释放，或在此翻车 |
| 通关/失败 | 满足/遗憾 | 通关=爽，失败="差1点"的赌狗遗憾 |

---

## 5. 关键设计原则

1. **骰子组合是主要伤害来源（70%+）**
   - 不能设计成"骰子只是资源，卡牌才是核心"
   - 玩家应该感受到：摇出好组合 = 战斗赢了一半

2. **卡牌是调味品，不是主菜**
   - 属性卡兜底，战斗卡放大，进化卡是大奖
   - 卡牌不能复杂到让玩家忽略骰子

3. **每局都有"神之一手"的可能**
   - 即使属性很低，摇出顺子+斩击联动也能翻盘
   - 低属性+好骰子 > 高属性+烂骰子（保持赌狗希望）

4. **重摇是心理博弈的核心**
   - 免费2次是"安全垫"，让玩家敢于赌
   - 重摇卡是"加码"，高手可以计算概率决定要不要搏

5. **观战不能无聊**
   - 自动战斗必须视觉效果强烈
   - 伤害数字、特效、进化动画是观战体验的核心

---

## 6. MVP 适配说明

| 功能 | MVP 实现 | 后续扩展 |
|------|----------|----------|
| 骰子组合 | 仅三条、顺子、对子 | 葫芦、四条、同花顺 |
| 组合特效 | 简单光效+数字放大 | 全屏特效、镜头震动 |
| 站位 | 前中后排固定效果 | 格子Buff、阵型加成 |
| 进化觉醒 | 仅解锁形态，无动画 | 变身动画、技能变更 |
| 骰子面升级 | 仅数字增减 | 元素效果（冰火雷） |
