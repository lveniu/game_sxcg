## 可立即分配的任务（无后端阻塞）

### FE-13：商店UI完善

**现状**：ShopPanel 已存在但功能基础，缺少等级/稀有度/限购等交互。

**增强内容**：

```
布局：
┌──────────────────────────────────┐
│  🏪 商店 Lv.2                    │  商店等级（随关卡提升）
│  💰 金币: 320                    │  玩家金币
│  刷新: 20💰                      │  刷新费用
├──────────────────────────────────┤
│  ┌────┐ ┌────┐ ┌────┐ ┌────┐   │
│  │★★★│ │★★ │ │★  │ │🔒 │   │  商品卡片
│  │神器│ │精良│ │普通│ │锁定│   │  稀有度标识
│  │100💰│ │50💰│ │20💰│ │Lv3│   │  价格 / 等级锁定
│  └────┘ └────┘ └────┘ └────┘   │
│  [已购]  [购买]  [购买]  [锁定]   │  按钮状态
├──────────────────────────────────┤
│  [刷新商品]        [关闭商店]     │
└──────────────────────────────────┘
```

**功能清单**：
1. **商品稀有度视觉**：
   - 普通（白框）、精良（绿框+发光）、稀有（蓝框+粒子）、史诗（紫框+光环）
   - 稀有度边框颜色 + 卡片底部稀有度条
2. **商店等级显示**：
   - 标题区显示当前商店等级（Lv.1/2/3）
   - 等级越高，稀有商品概率越高（用后端接口，前端只展示）
3. **限购机制**：
   - 已购买商品灰化 + "已购" 标签
   - 每个商品右上角限购数量标记（如 "1/1"）
4. **等级锁定**：
   - 高等级商品显示 🔒 + "Lv.3解锁"
   - 等级不足时购买按钮禁用，点击提示 "商店等级不足"
5. **刷新交互**：
   - 刷新按钮显示费用（如 "🔄 刷新 20💰"）
   - 金币不足时按钮禁用
   - 刷新时商品卡片旧→缩小消失→新→放大出现（0.3s交错）
6. **购买动画**：
   - 点击购买 → 商品卡片飞向背包图标（DOTween路径动画）
   - 金币数字 -N 飘字（红色向上）
   - 卡片变为已购状态（灰化+对勾）

### FE-14：装备套装UI

**现状**：EquipPanel 已有完整的装备管理功能，需要增加套装效果展示。

**增强内容**：

```
装备面板（增强后）：
┌──────────────────────────────────┐
│ 英雄列表 │  装备槽位 + 套装效果    │
│ ⚔战士 ★ │  武器：烈焰之剑        │
│          │  防具：铁盾            │
│          │  饰品：空              │
│          │                        │
│          │  🛡 套装效果：          │
│          │  ██████░░ 铁壁(2/4)    │
│          │  ✅ 2件：防御+20%      │  已激活（金色高亮）
│          │  🔒 4件：受击反伤15%   │  未激活（灰色）     │
│          │                        │
│          │  属性对比：             │
│          │  ⚔ 45 → 55 (+10)      │
└──────────────────────────────────┘
```

**功能清单**：
1. **套装进度条**：已装备件数 / 总件数（如 2/4）
2. **套装效果分两行**：
   - 已激活效果：金色文字 + ✅ + 绿色背景
   - 未激活效果：灰色文字 + 🔒
3. **装备替换时实时更新套装状态**：
   - 穿上套装件 → 进度+1 → 达到阈值时效果激活动画（闪光+飘字"铁壁套装激活！"）
   - 卸下套装件 → 进度-1 → 效果取消（红色闪烁提示）
4. **装备选择列表标识**：
   - 属于某套装的装备带套装色标（如铁壁=蓝标，狂战=红标）
   - Tooltip中显示"铁壁套装 2/4"

### FE-15：卡组编辑UI

**现状**：无卡组编辑界面。需要新建 CardDeckEditorPanel.cs。

**布局设计**：

```
┌──────────────────────────────────────┐
│  🃏 卡组编辑                 [关闭]  │
├──────────────────────────────────────┤
│                                      │
│  当前卡组 (8/12)                      │
│  ┌────┐┌────┐┌────┐┌────┐          │
│  │🔥  ││🛡  ││❄️  ││⚡  │ ← 可拖拽 │
│  │火球 ││盾墙 ││冰冻 ││闪电 │  重排   │
│  │★★ ││★  ││★★★││★★ │          │
│  └────┘└────┘└────┘└────┘          │
│  ┌────┐┌────┐┌────┐┌────┐          │
│  │🗡  ││💚  ││🎯  ││🌀  │          │
│  │连击 ││治疗 ││精准 ││旋风 │          │
│  │★  ││★★ ││★  ││★★ │          │
│  └────┘└────┘└────┘└────┘          │
│                                      │
│  ─── 卡池（可加入的卡牌）───          │
│  ┌────┐┌────┐┌────┐┌────┐          │
│  │🔥  ││💀  ││🛡  ││❄️  │          │
│  │陨石 ││毒雾 ││圣盾 ││暴风 │          │
│  │★★★││★★ ││★★★││★★ │          │
│  └────┘└────┘└────┘└────┘          │
│                                      │
│  [拖入卡组] / [拖出移除]              │
└──────────────────────────────────────┘
```

**功能清单**：
1. **上下分区**：当前卡组（已选卡牌） + 卡池（未选卡牌）
2. **拖拽排列**：
   - 卡组内拖拽重排顺序（影响战斗中出牌顺序）
   - 从卡池拖入卡组（添加）
   - 从卡组拖出（移除回卡池）
3. **卡牌详情Tooltip**：
   - 悬停显示：名称、描述、效果、稀有度、来源
4. **卡牌数量限制**：
   - 上限12张，满时卡池卡片不可拖入
   - 卡组数量指示器 "8/12"
5. **稀有度视觉**：
   - 普通=白边、精良=绿边+微光、稀有=蓝边+流光、史诗=紫边+粒子
6. **卡牌搜索/筛选**（可选，加分项）：
   - 按类型（攻击/防御/辅助）筛选
   - 按稀有度筛选

**交互细节**：
- 拖拽开始：卡片放大1.2x + 半透明
- 拖拽悬停在目标区：目标区高亮边框
- 放下成功：卡片缩小到目标位置（0.2s）
- 放下失败：卡片弹回原位（OutBack）
- 卡组满时拖入：红色闪烁 + Toast提示"卡组已满"

**预估**：+350行，新建 CardDeckEditorPanel.cs
