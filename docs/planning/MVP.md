# MVP 范围 — GameSXCG

> 最小可行产品：证明核心循环可玩。预估 **2~3 周** 完成。

## 原则

- 只做核心循环，不做局外 Meta
- 内容量极少，但要能让玩家感受到"掷骰子→出牌构筑→看自动战斗"的乐趣
- 所有"后续扩展"的功能全部剔除

---

## 1. 内容量

### 1.1 英雄
- **数量**: 1 个（战士）
- **进化**: 1 条直线（战士 → 狂战士），无分支
- **属性**: 生命、攻击、防御、攻击速度
- **技能**: 普攻 + 1 个主动技能（狂战士状态解锁）

### 1.2 敌人
- **小怪 x2** — 普攻型，血少防低
- **精英 x1** — 高攻低血，带简单技能
- **Boss x1** — 高血高防，2 个技能

### 1.3 卡牌（共 5 张）
| 名称 | 类型 | 效果 |
|------|------|------|
| 力量训练 | 属性 | 永久 +2 攻击 |
| 坚固护甲 | 属性 | 永久 +2 防御 |
| 斩击 | 战斗 | 消耗一个骰子，本场攻击+50% |
| 重摇 | 战斗 | 消耗一个骰子，重新掷该骰子 |
| 血性觉醒 | 进化 | 消耗三个相同点数，解锁进化为狂战士 |

### 1.4 骰子
- **数量**: 3 个，每个 6 面（1~6）
- **重摇**: 每局免费重摇 2 次（广告重摇 MVP 不做）
- **组合奖励**: MVP 只做最简单的
  - 三条（三个相同点数）— 攻击+10%
  - 顺子（5个连续）— 攻击速度+10%
  - 更复杂的组合留到 Alpha 阶段

### 1.5 战场
- **大小**: 固定 3×4，不可调节
- **格子 Buff**: 暂时不做，所有格子平等
- **单位上限**: 我方 1 个，敌方最多 3 个

---

## 2. 单局流程

```
[开始] → 选择战士（只有一个选项）
    ↓
[小怪战] → 掷骰子→出牌→自动战斗→获得 1 张随机卡牌
    ↓
[小怪战] → 同上
    ↓
[精英战] → 同上，获得 1 张紫卡（必出进化卡）
    ↓
[商店] → 简单商店（用战斗获得的金币买卡牌，MVP 只做随机三选一）
    ↓
[小怪战]
    ↓
[Boss战] → 通关或团灭
```

---

## 3. 系统范围

### ✦ 必做（在循环内）
- [ ] 骰子随机生成和重摇
- [ ] 卡牌出手和效果计算
- [ ] 自走棋自动战斗（普攻+技能AI）
- [ ] 单个单位的进化变身（战士→狂战士）
- [ ] 简单的等级节点跳转（标准的 Roguelike 节点地图，MVP 可以是线性的）
- [ ] 战斗结算和奖励发放

### ✖ 不做（MVP 剔除）
- 局外成长（天赋树、图鉴等）
- 多英雄和职业分支
- 观看广告功能
- 棋盘格子 Buff
- 复杂的骰子组合奖励
- 音效和美术包装（先用程序代替）
- 存档系统（MVP 单局即终）

---

## 4. 验收标准

MVP 完成后，玩家应该能够：
1. 掷出骰子，看到点数和简单组合识别
2. 根据点数出手卡牌，感受到构筑影响
3. 看自走棋自动战斗，战士按构筑打出不同输出
4. 体验到进化为狂战士后战斗方式的变化
5. 从开始到 Boss 打完一整局
