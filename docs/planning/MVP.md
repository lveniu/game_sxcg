# MVP 范围 — GameSXCG

> 最小可行产品：证明核心循环可玩。预估 **2~3 周** 完成。

## 原则

- 只做核心循环，不做局外 Meta
- 内容量极少，但要能让玩家感受到"掷骰子→出牌构筑→看自动战斗"的乐趣
- 所有"后续扩展"的功能全部剔除

---

## 1. 内容量

### 1.1 英雄（3 个职业）
| 职业 | 定位 | 进化路线 | 特点 |
|------|------|----------|------|
| 坦克 | 高防高血 | 链甲使者 | 吸收伤害、护盾反弹 |
| 射手 | 远程输出 | 巡游射手 | 高暴击、越远越痛 |
| 刺客 | 爆发凸进 | 影舞者 | 高移动速度、闪避、背刺 |

- MVP 每个职业只有 **1 条直线进化**，无分支
- 属性：生命、攻击、防御、速度、暴击率
- 技能：普攻 + 1 个主动技能（进化后解锁/增强）

### 1.2 敌人（5 种）
- **小怪 x3** — 普攻型、自爆型、高血型
- **精英 x1** — 混合型，高攻 + 简单技能
- **Boss x1** — 高血高防，2 个技能，有阶段转变

### 1.3 卡牌（共 8 张）
| 名称 | 类型 | 效果 |
|------|------|------|
| 力量训练 | 属性 | 永久 +3 攻击 |
| 坚固护甲 | 属性 | 永久 +3 防御 |
| 灵敏训练 | 属性 | 永久 +2 速度 |
| 斩击 | 战斗 | 消耗一个骰子，本场攻击+50% |
| 重摇 | 战斗 | 消耗一个骰子，重新掷该骰子 |
| 护盾冲击 | 战斗 | 消耗两个骰子，获得护盾并冲撞敌人 |
| 寻找弱点 | 战斗 | 消耗一个骰子，本场暴击率+30% |
| 进化觉醒 | 进化 | 消耗三个相同点数，解锁进化 |

### 1.4 骰子
- **数量**: 3 个，每个 6 面（1~6）
- **重摇**: 每局免费重摇 2 次
- **组合奖励**: MVP 只做最简单的
  - 三条（三个相同点数）— 攻击+10%
  - 顺子（5个连续）— 攻击速度+10%
  - 更复杂的组合留到 Alpha 阶段

### 1.5 战场
- **大小**: 固定 3×4，不可调节
- **格子 Buff**: 暂时不做，所有格子平等
- **单位上限**: 我方 1 个，敌方最多 3 个

## 2. 单局流程

```
[开始] → 选择职业（坦克 / 射手 / 刺客）
    ↓
[小怪战] → 掷骰子→出牌→自动战斗
    ↓
│── 战胜 → 获得金币+随机卡牌 → 进入下一关
│── 战败 → 根据已通关节点数给予临时奖励 → 结束本局
    ↓
[小怪战] → 同上
    ↓
[精英战] → 同上，获得 1 张紫卡（必出进化卡）
    ↓
[商店] → 简单商店（用金币买卡牌，MVP 只做随机三选一）
    ↓
[小怪战]
    ↓
[Boss战] → 通关或团灭 → 结算奖励
```

## 3. 系统范围

### ✦ 必做（在循环内）
- [ ] 骰子随机生成和重摇
- [ ] 卡牌出手和效果计算
- [ ] 自走棋自动战斗（普攻+技能AI）
- [ ] 单个单位的进化变身（战士→狂战士）
- [ ] 简单的等级节点跳转（标准的 Roguelike 节点地图，MVP 可以是线性的）
- [ ] 战斗结算和奖励发放

### ✖ 不做（MVP 剔除）
- 局外成长（天赋树、图鉴等）
- 观看广告功能
- 棋盘格子 Buff
- 复杂的骰子组合奖励
- 音效和美术包装（先用程序代替）
- 存档系统（MVP 单局即终）

---

## 4. 验收标准

MVP 完成后，玩家应该能够：
1. 选择坦克/射手/刺客中一个职业开局
2. 掷出骰子，看到点数和简单组合识别
3. 根据点数出手卡牌，感受到构筑影响
4. 看自走棋自动战斗，不同职业打出不同风格的战斗
5. 体验到进化后战斗方式的变化
6. 战胜继续往下，战败也能获得临时奖励
7. 从开始到 Boss 打完一整局
