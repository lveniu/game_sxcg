## 一、TapTap 首发方案概述

### 为什么选 TapTap 首发？

| 维度 | 说明 |
|------|------|
| **无需版号** | TapTap 不审核游戏版号，只审核内容合规，可立即上线 |
| **核心玩家聚集** | Roguelike/策略玩家最集中的平台，新游推荐效果好 |
| **社区生态强** | 开发日志、玩家互动、评分体系完整 |
| **审核快速** | 1-3天审核通过 |
| **零门槛发布** | 可先发布预约页面，不依赖完整包体 |
| **Android APK** | 支持 WebGL 构建产物，技术对接成熟 |

### 与微信小游戏的对比

| 维度 | TapTap（首发） | 微信小游戏（备用） |
|------|---------------|------------------|
| 版号要求 | ❌ 不需要 | ✅ 必须 |
| 审核周期 | 1-3天 | 1-7天 |
| 包体限制 | 无硬性限制 | 4MB主包+20MB子包 |
| 社交裂变 | ❌ 无 | ✅ 强（朋友圈/群） |
| 核心玩家 | ✅ Roguelike聚集地 | ❌ 偏休闲 |
| 获客成本 | 中 | 低 |
| 上线速度 | 快（立即） | 慢（版号卡3-6个月） |

## 六、技术对接需求（给前端/后端）

| 要求 | 说明 |
|------|------|
| 构建目标 | WebGL Android APK |

| 功能 | 必要性 | 说明 |
|------|--------|------|

| 云存档 | 可选 | 多设备同步 |
