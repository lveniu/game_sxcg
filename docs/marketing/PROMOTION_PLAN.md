## 一、定位与核心卖点

### 定位一句话
**「首款骰子组合+卡牌构筑+自动棋盘的竖屏Roguelike小游戏，三分钟一局，策略不重样」**

### 核心卖点（优先级排序）

| 优先级 | 卖点 | 一句话描述 | 目标受众 |
|--------|------|-----------|---------|
| 🔴 P0 | **骰子赌狗快感** | 摇出三条/顺子→看组合在战场爆炸，70%爽感来自掷骰那一刻 | 核心玩家 |
| 🔴 P0 | **肉鸽三选一奖励** | 每局通关后选新骰子面/属性强化/额外重摇，局局不同 | 核心玩家 |
| 🟡 P1 | **遗物策略层** | 收集遗物触发被动效果，与骰子组合形成Build流派 | 核心玩家 |
| 🟡 P1 | **骰子面升级** | 骰子每面可定制升级（6→闪电/3→冰冻），打造专属骰子 | 核心玩家 |
| 🟡 P1 | **Boss机制战** | 护盾互换/伤害反弹/狂暴等机制怪，策略挑战升级 | 核心玩家 |
| 🟢 P2 | **自动战斗观赛** | 摇完骰子就看你的组合碾压敌人，爽感释放 | 泛玩家 |
| 🟢 P2 | **三分钟一局** | 碎片时间可玩，地铁/厕所/等位都能来一局 | 泛玩家 |

### 爽感公式（宣传核心）
```
紧张（掷骰）→ 期待（看重摇）→ 释放（组合爆炸）→ 满足（碾压观战）
```

## 二、目标受众画像

### 主要受众：核心策略玩家（70%精力）

| 维度 | 描述 |
|------|------|
| **人群** | 18-35岁，玩过杀戮尖塔/暗黑地牢/小丑牌等Roguelike |
| **平台** | 微信小游戏、TapTap、B站游戏区 |
| **痛点** | 小游戏平台骰子Roguelike品质低，缺乏深度策略产品 |
| **触达** | 游戏评测博主、Roguelike社区、TapTap新游推荐 |
| **转化点** | 骰子组合+卡牌联动的策略深度，非换皮、有创新 |

### 次要受众：泛休闲玩家（30%精力）

| 维度 | 描述 |
|------|------|
| **人群** | 25-40岁，微信小游戏用户，碎片时间消遣 |
| **平台** | 微信朋友圈、微信群、抖音小游戏 |
| **痛点** | 无聊打发时间，想要有策略感但不肝 |
| **触达** | 微信社交裂变、短视频种草 |
| **转化点** | 三分钟一局、自动战斗不累、骰子运气好能翻盘 |

## 六、素材风格统一规范

基于UI设计文档的色彩规范：

| 元素 | 规范 |
|------|------|
| **背景色** | #2C3E50 深色背景（统一暗色调） |
| **骰子/金币** | #F5A623 金黄色（核心视觉锚点） |
| **伤害/危险** | #E74C3C 红色（爆炸感） |
| **成功/治愈** | #2ECC71 绿色（正向反馈） |
| **主按钮** | #4A90D9 蓝色（行动引导） |
| **品质色** | 白→蓝→紫→金（卡牌/遗物稀有度） |

**风格关键词**：暗色系、高对比、骰子金色、数字夸张、特效明显
