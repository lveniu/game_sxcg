# 海外上架备选方案

> **版本**: v1.0  
> **日期**: 2026-05-09  
> **状态**: 待审核确认

---

## 一、推荐市场：东南亚 → 日韩 → 欧美（递进策略）

### 为什么从东南亚起步？

| 维度 | 东南亚 | 日韩 | 欧美 |
|------|--------|------|------|
| **游戏偏好** | 偏好轻度策略/棋牌类，骰子玩法接受度高 | 偏好数值深度，Roguelike核心玩家多 | 偏好机制创新，Roguelike成熟市场 |
| **竞品环境** | 中低品质占多数，品质窗口期 | 品质要求极高，竞品成熟 | 极度饱和，获客成本极高 |
| **语言** | 英文化即可覆盖，本地化成本低 | 必须本地化（日/韩文） | 英文化即可，多语言可选 |
| **合规** | 无需版号，审核宽松 | 需当地分类评级（如CERO） | 需ESRB/PEGI评级 |
| **获客成本** | $0.3-0.5/次安装 | $0.5-1.0/次安装 | $2-5/次安装 |
| **支付能力** | ARPU较低（$0.5-2/月） | 中等（$2-5/月） | ARPU较高（$5-15/月） |
| **启动难度** | ⭐⭐ 低 | ⭐⭐⭐⭐ 高 | ⭐⭐⭐⭐⭐ 极高 |

**建议**：东南亚首站，验证海外接受度后再拓展日韩/欧美。

## 二、海外发布渠道

### 2.1 微信小游戏海外版
| 项目 | 说明 |
|------|------|
| 平台 | 微信海外版（WeChat Mini Games） |
| 覆盖 | 东南亚（新加坡/马来西亚/印尼/泰国）用户基数大 |
| 优势 | 社交裂变机制与国内一致，无需额外获客 |
| 要求 | 无版号要求（海外版），需企业微信开发者账号 |
| 审核 | 1-3个工作日，内容合规审查较
### 2.2 Steam（PC版）
| 项目 | 说明 |
|------|------|
| 平台 | Steam Greenlight |
| 覆盖 | 全球，核心Roguelike玩家最集中的平台 |
| 优势 | 社区口碑传播力强，评测体系成熟 |
| 要求 | $100上架费，无版号要求 |
| 审核 | 自动审核+内容合规审查（3-7天） |

### 2.3 独立游戏平台
| 平台 | 说明 | 适合时机 |
|------|------|---------|
| Itch.io | 免费发布，独立游戏社区 | 早期版本/Demo测试 |
| Kongregate | HTML5小游戏平台 | 微信小游戏前做技术验证 |
| CrazyGames | 浏览器游戏聚合 | 获取海外浏览器玩家 |

## 三、海外上架流程（东南亚首站）

```
企业微信海外开发者账号注册(1天)
    ↓
本地化准备（英文化+文化审查）(3-5天)
    ↓
海外版素材准备（截图/描述/宣传图）(2-3天)
    ↓
上传海外版本(1天)
    ↓
提交审核(1-3天)
    ↓
审核通过 → 上线东南亚
    ↓
数据收集 → 优化 → 拓展日韩/欧美
```

## 四、英文化清单

### 4.1 需要翻译的内容
| 内容 | 预估字数 | 优先级 |
|------|---------|--------|
| 游戏UI界面（菜单/按钮/标签） | ~500字 | P0 |
| 游戏内文本（卡牌描述/骰子效果/遗物描述） | ~3000字 | P0 |
| 成就系统文本 | ~500字 | P1 |
| 游戏描述（商店页） | ~500字 | P0 |
| 隐私政策 | ~1000字 | P0 |
| 营销文案（TapTap/B站/社群） | 按渠道 | P1 |
| 教程文本 | ~300字 | P0 |
| 错误提示/Toast | ~50条 | P0 |

### 4.2 文化审查要点
| 类型 | 注意 |
|------|------|
| 宗教/文化敏感 | 避免特定宗教符号或敏感历史 |
| 赌博暗示 | "骰子"本身可能被视为赌博元素，UI需强调"策略"而非"赌博" |
| 暴力内容 | 战斗描述避免过于血腥 |
| 数字/颜色 | 部分地区对数字4有忌讳（华人市场除外） |

## 五、商业化调整（海外）

### 5.1 收入模式对比
| 维度 | 国内 | 东南亚 | 日韩 | 欧美 |
|------|------|--------|------|------|
| 激励视频ARPU | $0.8-1.2/DAU | $0.3-0.6/DAU | $0.5-1.0/DAU | $1.5-3.0/DAU |
| 内购渗透率 | 8-15% | 3-8% | 5-12% | 5-10% |
| ARPU目标 | $5-8/月 | $2-4/月 | $3-6/月 | $8-15/月 |
| 主要付费点 | 通行证+广告+皮肤 | 广告为主 | 通行证+抽卡 | 通行证+DLC |

### 5.2 海外付费点设计
| 付费点 | 国内版 | 海外版调整 |
|--------|--------|-----------|
| 激励视频 | 重摇次数 | 重摇次数+跳过战斗 |
| 赛季通行证 | ¥18-38/月 | $2.99-4.99/月 |
| 骰子皮肤 | ¥6-18/个 | $0.99-2.99/个 |
| 额外重摇 | ¥3-8/次 | $0.49-0.99/次 |
| **新增** | 无 | 移除直接抽卡付费（海外合规风险高） |

## 六、时间线与并行策略

### 6.1 推荐并行路径
```
国内路线                          海外路线
企业微信开发者注册 → 版号申请     企业微信海外注册 → 英文化
    ↓                               ↓
    ↓（3-6个月版号周期）             海外版上传 → 审核上线（2-4周）
    ↓                               ↓
    ←─────────  先发海外验证 → 反馈优化 → 反哺国内
```

### 6.2 关键时间节点
| 时间 | 国内 | 海外（东南亚首站） |
|------|------|-------------------|
| M0 | 企业微信注册 | 海外开发者注册 |
| M1 | 隐私政策撰写 | 英文化启动 |
| M2 | 隐私政策审核 | 本地化完成+测试 |
| M3 | 海外版上线（如先做海外） | 上线东南亚 |
| M4 | 版号提交 | 数据收集+优化 |
| M5-10 | 版号审批中 | 拓展日韩/欧美 |

## 七、风险评估

| 风险 | 等级 | 应对 |
|------|------|------|
| 语言翻译质量差 | 🟡中 | 使用专业翻译工具+人工校对，避免机翻感 |
| 文化不适配 | 🟡中 | 上架前做目标市场文化审查 |
| 支付渠道不通 | 🟢低 | 微信海外版支持多种当地支付方式 |
| 海外用户量小 | 🟡中 | 先发东南亚，用户基数大，验证后再扩 |
| 国内海外版本差异 | 🟡中 | 核心代码复用，仅换UI/文本/支付 |

## 八、建议

### 立即可做
1. **企业微信海外开发者注册** — 准备材料，与微信海外版联系
2. **英文化启动** — 将游戏内所有文本提取为可翻译的键值对
3. **文化审查** — 审查骰子元素在海外市场的合规性

### 中期（版号申请后）
1. 海外版正式上架
2. 东南亚TapTap社区运营
3. 收集海外用户数据，优化留存/变现

### 长期（产品成熟后）
1. 拓展日韩市场（本地化日/韩文）
2. Steam PC版上线
3. 欧美市场（付费DLC/通行证模式）

---

> **一句话总结**：东南亚首站、英文化先行、微信海外版+TapTap双渠道。先用最小成本验证海外接受度，再决定是否投入日韩/欧美。
