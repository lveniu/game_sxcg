## 一、建号步骤

### 1.1 开发者入驻
1. 访问 [tap.cn](https://tap.cn) 注册账号
2. 进入开发者中心 → 开发者入驻
3. 选择企业主体（与微信公众号主体一致）
4. 上传营业执照 → 等待认证（通常1-3工作日）
5. 企业认证通过后获得开发者权限

### 1.2 创建游戏条目
1. 开发者中心 → 我的游戏 → 创建新游戏
2. 填写基本信息：
   - 游戏名称（与微信小游戏一致或相近）
   - 游戏类型：Roguelike / 策略 / 棋牌
   - 支持平台：iOS / Android / 微信小游戏
   - 封面图（建议 400×400 横图）
   - 游戏截图（上传 5-10 张，1080×1920 竖屏）
   - 游戏描述（100字以内，核心卖点提炼）
   - 开发团队名称
   - 联系方式

### 1.3 发布预约页面
1. 游戏创建审核通过后（通常1-3天）
2. 可先发布"预约中"页面
3. 积累预约用户 → 上线时推送通知
