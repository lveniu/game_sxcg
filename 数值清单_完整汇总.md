## 四、伤害公式 (GameBalance.CalculateDamage)

```
暴击判定: Random.value < critRate → 暴击
暴击倍率: isCrit ? (1.5 + critDamage) : 1.0
原始伤害: attackerAtk × damageMultiplier × critMult
最终伤害: Max(1, Round(rawDamage) - targetDef)
```

| 参数 | 说明 |
|------|------|
| 基础暴击倍率 | 1.5x |
| 暴击伤害默认值 (BattleCritDamage) | 0.5 (=50%额外) → 总暴击倍率2.0x |
| 最低伤害 | 1 |

### 治疗公式
```
公式: Round(baseHeal × healMultiplier)
```

| 英雄属性 | Round(base × starMult) + equip |
